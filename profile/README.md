## Projects

### [caboose-mcp](https://github.com/caboose-mcp/caboose-mcp)
The caboose-mcp monorepo. Contains the core MCP server and a VS Code extension for integrating AI-powered tooling directly into your editor workflow.

### [meml](https://github.com/caboose-mcp/meml)
**Emoji Markup Language (EML)** — a config language with first-class emoji support. A Go-based parser and toolchain for a human-friendly configuration format.
